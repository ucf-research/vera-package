vera-xr.io/demo

The VERA Sandbox Demo is a simple mock experiment designed to showcase the tools VERA provides in a hands-on environment. To get the most out of this sandbox demo, please use Unity version 6000 or up, and the most recent version of the VERA Unity package.

See the VERA Sandbox Demo documentation for more details on how to use this demo: vera-xr.io/demo. Upon fresh import, this sandbox demo does *NOT* have VERA fully integrated. Follow the steps in the documentation to fully transform this experiment to use VERA.
